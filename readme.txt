【守って！猫騎士様！】

ネズミ軍団に襲われたお姫様を猫騎士を操作して守り切れ！

【操作説明】
・キーボード入力
	移動　WASDキー
	攻撃　スペースキー
	魔法の発動　Bキー
	姫と手をつなぐ　Nキー

・ゲームパッド
	移動　左スティック
	攻撃　Bボタン
	魔法の発動　Yボタン
	姫と手をつなぐ　Xボタン

【力を入れたところ】
私が一番力を入れたところはゲーム全体の雰囲気作りです。
モデルからモーションまで自作し、可愛らしく取っつきやすい世界観を目指しました。
次に工夫した点は、姫と手をつなぎ移動できるようにしたことです。
タワーディフェンスゲームでありながら守る対象と一緒に行動できるようにし、よりクリアし易くしました。
他にも、ポリゴンを利用した剣の軌跡の表示や、ダメージやマヒ状態での色変化など
視覚的な分かりやすさを大切にしました。
また、モデルのロードや敵の出現位置管理にはテキストデータを使い、変更を手軽にできるようにしました。
