このゲームは主人公である猫の騎士を操作し、マップの中央にいる猫のお姫様を次々と迫りくるネズミ軍団から守り切るというというものです。
中でも私が一番こだわったのは、ゲーム全体の雰囲気づくりです。モデルもモーションもすべて自作し、ゲーム初心者でも遊びやすいように難易度を考え、ゆるくて可愛らしい雰囲気を壊さないように気を付けました。
初めて制作した３Dゲームで、ポリゴンの表示などの基礎の基礎から始めました。
最初はモデルに動きをつけるのにも一苦労でしたが、テキストファイルからモーションの読み込みを行えるようにし、より手軽に読み込めるようになりました。
他にも、ポリゴンメッシュを利用し武器に軌跡をつけたり、攻撃を受けた際に赤く点滅する処理を入れたり視覚的にもわかりやすさを追求して制作しました
